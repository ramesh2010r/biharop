# Visual Assets Creation Guide - Bihar Opinion Poll Blogs

**Document Purpose**: Complete specifications for creating all visual assets (images, charts, infographics, maps) for blog posts  
**Design Tool**: Canva Pro, Figma, Adobe Photoshop, or similar  
**Target**: Professional, data-rich, bilingual (Hindi-English) visuals

## ðŸŽ¨ Brand Guidelines

### Color Palette
```
Primary Colors:
- Orange: #FF6B35 (BJP saffron inspired)
- Blue: #004E89 (Congress blue inspired)
- Green: #00A878 (RJD green inspired)
- Saffron: #FF9933 (National flag color)

Secondary Colors:
- Light Orange: #FFB88C
- Light Blue: #6FBBE9
- Light Green: #8FD9C4
- Gray: #495057
- Light Gray: #F8F9FA

Text Colors:
- Heading: #212529 (dark gray)
- Body: #495057 (medium gray)
- Accent: #FF6B35 (orange)
```

### Typography
```
Hindi Font: Noto Sans Devanagari (Bold, SemiBold, Regular)
English Font: Poppins (Bold, SemiBold, Regular)

Heading Sizes:
- H1: 48px (Bold)
- H2: 36px (SemiBold)
- H3: 28px (SemiBold)
- Body: 18px (Regular)
- Caption: 14px (Regular)
```

### Logo Placement
- Always include "Opinion Poll Bihar" logo watermark in bottom-right corner
- Logo opacity: 50% for watermark
- Logo size: 150x60px
- Logo position: 20px margin from edges

## ðŸ› ï¸ Creation Tools & Resources

### Recommended Design Tools
1. **Canva Pro** (easiest, templates available)
   - Pros: Quick, templates, bilingual support
   - Cons: Limited customization
   - Best for: Featured images, social cards

2. **Figma** (professional, collaborative)
   - Pros: Professional, free, collaborative
   - Cons: Learning curve
   - Best for: Infographics, complex layouts

3. **Adobe Photoshop** (most powerful)
   - Pros: Complete control, best quality
   - Cons: Expensive, complex
   - Best for: Photo editing, complex graphics

4. **Chart.js or Google Charts** (for web charts)
   - Pros: Interactive, responsive, data-driven
   - Cons: Requires coding
   - Best for: Live charts on website

### Free Resources
- **Icons**: Flaticon.com, FontAwesome
- **Fonts**: Google Fonts (Noto Sans Devanagari, Poppins)
- **Stock Photos**: Unsplash, Pexels
- **Color Palettes**: Coolors.co
- **Maps**: OpenStreetMap, MapChart

## ðŸš€ Quick Start Guide

### Step 1: Open Canva Pro
1. Create custom size: 1200x630px
2. Choose "Blank" template
3. Set background gradient (orange to blue)

### Step 2: Add Bihar Map
1. Search "India Map" in elements
2. Crop to show only Bihar
3. Add district boundaries using lines
4. Label key districts

### Step 3: Add Text
1. Title box: Hindi text, 48px, bold, white
2. Subtitle: Stats, 24px, semi-bold
3. Ensure text readable with shadow/glow

### Step 4: Add Branding
1. Upload Opinion Poll logo
2. Place in bottom-right corner
3. Set opacity to 50%

### Step 5: Export
1. Download as JPG
2. Optimize with TinyPNG
3. Upload to `/public/images/blog/`
